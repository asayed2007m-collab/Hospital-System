​🏥 Hospital Management System (Core Module)
​A comprehensive desktop application designed to manage the primary operations of a hospital, focusing on the relationship between Medical Staff and Patients. Built with C++ and Qt Framework, this project demonstrates advanced data handling and GUI interaction.
​🧠 Software Architecture & OOP
​This system is built on solid programming foundations:
​Object Relationships: Implemented a dynamic link between Doctor and Patient entities. When a doctor is added, they automatically appear as an option during patient registration.
​Encapsulation: Used private data members for IDs, names, and medical records to ensure data integrity.
​Search Algorithms: Integrated a search engine that filters through patient data using unique IDs to retrieve records instantly.
​🚀 Main Modules
​Doctor Registry: Add and manage the list of available specialists.
​Patient Admissions: Register patients and assign them to specific doctors from the existing database.
​Integrated Viewers: Dedicated table views for both doctors and patients to monitor hospital capacity.
​Quick Search: A specialized tool to find patient details and their assigned doctor by ID.
​🛠️ Tech Stack
​Language: C++ (Focus on Data Relationships)
​Framework: Qt (Widgets, Layouts, Signal/Slots)
​IDE: Visual Studio / Qt Creator
